# 原子操作
一个原子类型我们可以查看一个类型是不是线程安全的，通过`std::atomic::is_lock_free`
比如：
```c++
#include <atomic>
#include <iostream>
  
std::atomic<int> a;

int main(){

    a = 0;
    std::cout << a.is_lock_free() << std::endl;
    return 0;
}
```
如果返回1， 则说明这个类型在当前硬件上是原子的，也就是所有操作无需加锁，如果返回false，则需要我们手动添加锁结构

但是有一个原子类型没有`is_lock_free`函数，就是`std::atomic_flag`，这是一个原子标记，这个类型在初始化时会被自动的
